    Cascading Style Sheet (CSS)

    Cascading Style Sheet adalah standar W3C untuk mendefinisikan penyajian (presentation) dokumen yang ditulis dalam HTML. Penyajian ini mengacu pada cara dokumen disampaikan kepada pengguna, apakah di layar komputer, ditampilkan di ponsel, dll.

    Jenis-Jenis CSS Selector

    Selector Arti Contoh Penjelasan
    Universal Selector Menerapkan gaya ke semua elemen dalam dokumen * {} Menargetkan semua elemen dalam halaman
    Type Selector Menyesuaikan nama elemen h1, h2, h3 {} Menargetkan elemen <h1>, <h2>, dan <h3>
    Class Selector Menargetkan elemen yang memiliki atribut class sesuai dengan nama yang diberikan setelah titik (.) .note {} Menargetkan semua elemen yang memiliki class="note"
    p.note {} Menargetkan hanya elemen <p> dengan class="note"
    ID Selector Menargetkan elemen yang memiliki atribut id sesuai dengan nama yang diberikan setelah tanda pagar (#) #introduction {} Menargetkan elemen dengan id="introduction"
    Child Selector Menargetkan elemen yang merupakan child selector. Artinya cari elemen <a> yang merupakan child. dari elemen tag <li> li > a {} Menargetkan semua elemen <a> yang langsung berada dalam elemen <li>
    Descendant Selector Menargetkan elemen yang merupakan keturunan dari elemen lain (tidak harus anak langsung). Artinya dalam contoh tersebut adalah, semua elemen tag <a> adalah keturunan dari elemen tag <p> p a {} Menargetkan semua elemen <a> yang berada di dalam elemen <p>
    Adjacent Sibling Selector Menargetkan elemen yang merupakan saudara kandung terdekat setelah elemen lain h1 + p {} Menargetkan elemen <p> pertama setelah <h1>
    General Sibling Selector Menargetkan elemen yang merupakan saudara kandung umum. h1 ~ p {} Menargetkan semua elemen <p> yang merupakan saudara dari elemen <h1>

    Color (Warna)

    Properties color

    Properties color memungkinkan untuk mengatur warna teks (foreground color) dalam sebuah elemen. Hal ini dapat dilakukan dengan 3 cara:

    • RGB Values
    • Hexadecimal Code
    • Color Names

    RGB Values

    /* Using RGB values */

p {
  color: rgb(189, 189, 88);
}

    Hexadecimal Code

      /* Using hexa value */

  h2 {
      color: #f5deb3;
  }

    Color Names

      /* Using color name */

  h1 {
    color: white;
  }

    Properties background-color

    Properties background-color memungkinkan untuk mengatur warna background dalam sebuah elemen. Cara pengaturan warna nya sama seperti properties color.

    RBG Value

    body {
    background-color: rgb(200, 200, 200);
  }

    Hexadecimal Code

      h2 {
    background-color: #ee3e80;
  }

    Color Names

      p {
    background-color: white;
  }

    <!DOCTYPE html>
<html lang="en">

<head>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

        in!
    </p>
</body>

</html>

/* FOREGROUWND COLOR */

/* Using color name */

    color: white;
}

    color: #f5deb3;
}

    color: rgb(189, 189, 88);

    background-color: rgb(200, 200, 200);
}

    background-color: darkcyan;
}

h2 {
    background-color: #ee3e80;
}

    background-color: white;
}

    Properties ini memungkinkan mengatur tampilan jenis huruf (font) dalam sebuah halaman website.

    Properties font-size

    Properti ini digunakan untuk mengatur ukuran tulisan (font) dalam sebuah halaman website. Berikut ini beberapa cara untuk mengatur ukuran tulisan:

    • Pixels
    • Percentages
    • Ems

    Pixels

    body {
    font-family: Georgia, 'Times New Roman', Times, serif;
    font-size: 12px;
}

    Percentage

    h1,
h2 {
    font-family: Arial, Verdana, Geneva, sans-serif;
    font-size: 200%;
}

    Ems

    .credit {

    font-family: 'Courier New', Courier, monospace;
    font-size: 1.5em;
}

    Properties font-weight

    Properties ini digunakan untuk membuat tulisan bold (text bold). Properties ini memiliki dua nilai (value), yaitu normal dan bold. Untuk value normal maka tampilan teks akan terlihat normal, sedangkan untuk value bold maka tampilan teks akan terlihat tebal.

    properties font-weight

    .credit {

    font-family: 'Courier New', Courier, monospace;
    font-size: 1.5em;
    font-weight: bold;
}

    Properties text-transform

    Properties ini digunakan untuk mengubah huruf besar/kecil pada sebuah teks / tulisan.

    text-transform: uppercase

    body {
    font-family: Georgia, 'Times New Roman', Times, serif;
    font-size: 12px;
    text-transform: uppercase;
}

    text-transform: lowercase

    h1,
h2 {
    font-family: Arial, Verdana, Geneva, sans-serif;
    font-size: 200%;
    text-transform: lowercase;
}

    text-transform: capitalize

    .credit {

    font-family: 'Courier New', Courier, monospace;
    font-size: 1.5em;
    text-transform: capitalize;
}

    Properties text-decoration

    Properties ini digunakan untuk menambahkan dekorasi pada teks. Properties ini memiliki empat value (overline, underline, line through dan blink).

    HTML and CSS Code

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Text Properties</title>
    <link rel="stylesheet" href="text.css">
</head>
<body>
    <h1>This is Heading 1</h1>
    <h2>This is Heading 2</h2>
    <h3>This is Heading 3</h3>
    <h4>This is Heading 4</h4>
</body>
</html>

    h1 {
text-decoration: overline;
}

h2 {
text-decoration: line-through;
}

h3 {
text-decoration: underline;
}

h4 {
text-decoration: blink;
}
