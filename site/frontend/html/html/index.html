    HTML (Hypertext Markup Language)

    Hypertext Markup Language atau HTML adalah sebuah bahasa markup yang digunakan untuk mengatur struktur dari sebuah informasi yang ditampilkan di halaman web.

    Struktur HTML

    structure.html
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

    Berikut ini adalah struktur dari HTML5:

    <!DOCTYPE html> : Menjelaskan bahwa struktur yang digunakan berjenis HTML5.
<head>  : Berisikan metadata file HTML5.
<title> : Halaman judul web.
<body>  : Berisikan semua konten yang akan ditampilkan ke dalam web untuk pengguna.

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Elemen supscript</title>
</head>

<body>
    <p>
        This is supscript
    </p>
    E = MC<sup>2</sup>
</body>

</html>

    Output elemen <sup>

    Elemen supscript

    This is supscript

    E = MC2

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <p><s>Was $995</s></p>
    <p>Now only $375</p>

</body>

</html>

    Form

    <form>

    Tag elemen ini digunakan untuk mengumpulkan data / user input dalam sebuah website. Atribut action mendefinisikan ke mana data dikirim ketika user menekan tombol submit. Method adalah metode HTTP yang akan digunakan untuk mengirimkan data ke server.

    Penggunaan GET method dalam Form HTML

    • Digunakan untuk format pendek (short form) seperti search box
    • Retrieving data / meminta dari server.

    Penggunaan POST method dalam Form HTML

    • Mengupload file.
    • Mengirimkan data / form yang panjang.
    • Data sensitif (password, email, etc).
    • Menambah atau menghapus data dari sebuah database.

    6. maxlength="99": Menentukan maksimal jumlah karakter yang harus diisi.

    Select Input Attribute

    1. name: Atribute ini digunakan untuk mengelompokan checkbox input. Semua checkbox input harus memiliki value yang sama karena akan dianggap satu kelompok checkbox input. Hanya ada satu opsi dari checkbox input tersebut yang bisa dipilih dan dikirimkan ke server.

    3. value: Atribut ini menentukan isian opsi yang akan dipilih oleh penggunan dan value tersebut akan dikirimkan ke server. Setiap drop down list harus memiliki value yang berbeda agar server bisa mengetahui mana value yang dipilih oleh user.
